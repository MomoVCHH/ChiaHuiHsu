VR-Experiment-Design

I created the VR prototyp for the master thesis' experiment. It is written in C#. With Unity Engine and Google Resonance SDK and some Plugins, I created the virtual environment and audio effects (2D and 3D sound effects).And it was made according to the principles of gamification. Due to the visual and acoustic simulation design in the virtual environment, the experimental participants can immerse in another world.

Midi2Drum

Users only need to input their work's album profil, song's name, author's name, and a MIDI file into the generator. Through the conversion of the generator, the imported files will be automatically converted to the suitable file format which can be directly used in "Into The Rythm". Background Music and drum sounds will be analyzed, seperated and converted into wav format. The MIDI information will be converted with the help of TiMidity++-2.14.0 and a self-created XML parser written in Java.

Dancer Ball

The project is written in C++. The IDE Xcode and the open-source toolkit "open Frameworks" and the addon "Box2d" are needed. The recorded piano sounds are edited and used as sound clips. Users can freely draw golden lines through the computer keyboards so that they can interact with different circular objects

3D-interactive-program

Through the keys of the computer, the user can control the movement of the ball and its shape. This is a work made according to the reward and punishment mechanism. The user gets one point for collecting the correct items, and loses one point for the wrong objects.The goal is to get ten points.
