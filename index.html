VR-Experiment-Design

I created the VR prototyp for the master thesis' experiment. It is written in C#. For develop this work, the Unity Engine and Google Resonance SDK and some Plugins are used. This work contains visual and audio effects (2D and 3D sound effects), it is created based on the gamification principle.Due to the visual and acoustic simulation design in the virtual environment, the experimental participants can immerse in another world.


Midi2Drum

Users only need to input their work's album profil, song's name, author's name, and a MIDI file into the generator. Through the conversion of the generator, the imported files will be automatically converted to the suitable file format which can be directly used in "Into The Rythm". Background music and drum sounds will be analyzed, seperated and converted into wav format. The MIDI information will be converted with the help of TiMidity++-2.14.0 and a self-created XML parser written in Java.

2D interactive program

The project is written in C++. The IDE Xcode and the open-source toolkit "open Frameworks" and the addon "Box2d" are needed. The recorded piano sounds are edited and used as sound clips. Users can freely draw golden lines through the computer keyboards, so they can interact with different circular objects.




3D interactive program

Through the computer keyboards, the user can control the movement of the ball and its shape. This is a work made according to the reward and punishment mechanism. The user gets one point by the collecting of every right object and loses one point by the collecting of every wrong object. The goal is to get ten points.
