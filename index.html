VR-Experiment-Design

I created this VR prototype for an experiment in my master thesis. It is written in C#. For developing this work, I used the Unity Engine and Google Resonance SDK and some plugins. This work contains visual and audio effects (2D and 3D sound effects). It is created based on the gamification principle.Due to the visual and acoustic simulation design in the virtual environment, the experimental participants can immerse in another world.


Midi2Drum

Users only need to enter their work's album profil, song's name, author's name, and a MIDI file into the generator. Through the conversion of the generator, the imported files will be automatically converted to the suitable file format which can be directly used in "Into The Rythm". Background music and drum sounds will be analyzed, separated and converted into wav format. The MIDI information will be converted with the help of TiMidity++-2.14.0 and a self-created XML parser written in Java.

2D interactive program

This project is written in C++. I also used the IDE Xcode and the open-source toolkit "openFrameworks" and the addon "Box2d". The recorded piano sounds are edited and used as sound clips. Users can freely draw golden lines through the computer keyboards, so they can interact with different circular objects.




3D interactive program

Through the computer keyboards, the user can control the movement of the ball and its shape. This is a work made according to the reward and punishment mechanism. The user gets one point by the collecting a right object and loses one point by collecting a wrong object. The goal is to get ten points.




Supermarket
